Make new note* — `Cmd-n`  
\_Create note in new pane* — `Cmd-shift-n`  
*Open the link you’re hovering over in a new pane (while in edit mode)* — `Cmd-click`  
*Toggle edit/preview mode* — `Cmd-e`  
*Open quick switcher* — `Cmd-o`  
*Close active pane* — `Cmd-w`
