## Questions

-   Why is this slide important
-   What is the slide trying to say
-   What can be simplified
-   What should get attention
-   What looks intentional

## Conventions

-   One slide per message
-   Using images
-   If text is needed, use keywords, not text
-   What is the presentation's call to action
-   Show, don't tell
-   Don't overuse animations
-   Use consistent design
-   Use contrast, size, and color to set audience focus
-   Never more than six objects on a slide
-   Use a dark background
-   Structure
    -   Why
    -   Clear structure
    -   Rehearse
-   Structure
    -   Compelling intro
    -   Content
        -   Three key messages
    -   Conclusion
        -   Summarize the three key messages
-   SCQA
    -   Situation
    -   Complication
    -   Question
    -   Answer
