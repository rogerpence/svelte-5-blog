DataGate license manager version to DG version:

C.6 = Datagate 17.x
B.6 = Datagate 16.x
9.5 = DataGate 15.x
8.5 = DataGate 14.x
7.5 = DataGate 12.x
6.5 = DataGate 11.x
5.5 = DataGate 10.x
4.5 = DataGate 9.x
3.5 = DataGate 7.x or 8.x (same key/license schema)
2.5 = DataGate 5.0

For example,

This v3.5 license manager indicates that DG 7 or 8 is installed.
