-   Use F11 to toggle document preview (which hides all guides and shape outlines)
-   [Using footnotes and endnotes](https://affinityspotlight.com/article/take-control-of-footnotes-sidenotes-and-endnotes-in-affinity-publisher-2/)
-   When exporting Publisher docs to PDF, set the Export `Area` property dropdown to `All Pages` to create a PDF with a single pages (the default is to double pages horizontally). Very sadly, you have to do this each time you export to PDF.
